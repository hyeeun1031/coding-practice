# 매출이 높은 배우 찾기

DVD Rental Store 데이터베이스에는 어느 한 DVD 대여 프랜차이즈의 고객 정보, DVD 대여 정보, 제공 중인 영화의 정보 등이 들어있습니다. <br>1

대여 매출이 높았던 상위 5명의 배우를 추리고 특별 코너를 만들어 프로모션에 활용하려고 합니다. <br>
데이터베이스를 조회해 배우별 대여 매출 합계를 계산하고, 그 중 상위 5명의 정보를 출력하는 쿼리를 작성해주세요. <br>
배우별 대여 매출 합계는 배우가 출연한 작품들의 매출 합계입니다. <br>

쿼리 결과에는 아래 컬럼이 있어야 하며, 매출액 기준 내림차순으로 정렬되어 있어야 합니다. <br>

- first_name: 배우의 이름
- last_name: 배우의 성
- total_revenue: 배우 출연작들의 총 매출

### actor

| 컬럼명 | 데이터 타입 | 설명 |
|------|-----------|------|
| actor_id     | integer  | 배우 ID                  |
| first_name   | string   | 배우 이름 (이름)         |
| last_name    | string   | 배우 이름 (성)           |
| last_update  | datetime | 배우 데이터 업데이트 시점 |

### film_actor

| 컬럼명        | 타입     | 설명                     |
|--------------|----------|--------------------------|
| film_id      | integer  | 영화 ID                  |
| actor_id     | integer  | 배우 ID                  |
| last_update  | datetime | 출연 정보 업데이트 시점 |
