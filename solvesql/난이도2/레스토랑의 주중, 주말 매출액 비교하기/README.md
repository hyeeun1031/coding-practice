# 레스토랑의 주중, 주말 매출액 비교하기

Waiter's Tips 데이터베이스는 어떤 레스토랑의 영수증 데이터를 담고 있습니다. 영수증에는 식사 금액, 팁, 결제자의 성별 등 결제 관련 정보가 포함되어 있습니다. <Br>

레스토랑의 주인인 당신은 신규 직원 고용과 적절한 재료 수급을 위해 주중, 주말 각각의 매출액 규모를 파악하려고 합니다. <br>
데이터베이스를 조회해 주중, 주말의 합계 매출 규모를 집계하는 쿼리를 작성해주세요. 쿼리 결과에는 아래 컬럼이 있어야 하고, 매출액 합계 기준 내림차순으로 정렬되어 있어야 합니다. <br>

- week: 토/일요일의 경우 'weekend', 다른 요일의 경우 'weekday'
- sales: 매출액 합계

### tips

| 컬럼명       | 데이터 타입 | 설명                      |
|--------------|------------|---------------------------|
| total_bill   | number     | 식사 금액 ($)             |
| tip          | number     | 팁 ($)                    |
| sex          | string     | 결제자 성별               |
| smoker       | string     | 일행 중 흡연자 존재 여부  |
| day          | string     | 요일                      |
| time         | string     | 시간대                    |
| size         | integer    | 일행 수 (명)              |
